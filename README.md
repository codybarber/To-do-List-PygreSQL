#To-Do List

##Overview of Project
A to-do list created using Python, Flask, and PostgreSQL. Allows for the addition and removal of tasks, and striking through completed tasks.


##Technologies, Frameworks, and Programming Languages used
* HTML5 & CSS3
* Bootstrap 3
* Python
* Flask
* PostgreSQL
